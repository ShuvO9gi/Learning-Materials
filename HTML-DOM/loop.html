Different Kinds of Loops

JavaScript supports different kinds of loops:
for - loops through a block of code a number of times
for/in - loops through the properties of an object
for/of - loops through the values of an iterable object
while - loops through a block of code while a specified condition is true
do/while - also loops through a block of code while a specified condition is true

For loop

Loops can execute a block of code a number of times.
for(let i = 0; i < 10; i++) { //block of code }
you can use this loop in an iterable object(like array, string) for accessing all the item by their index number. it can also useful for calculation of large number.
iterable - An object that has iterable properties.
Iterating - It simply means looping over a sequence of elements(iterable object).

For Of Loop

The JavaScript for of statement loops through the values of an iterable object(such as Arrays, Strings, Maps, Sets, NodeLists, and more).

While Loop

The while loop loops through a block of code as long as a specified condition is true. It just like for loop with first and last parameter ommited. Functionality of while loop & for loop is almost same.

while(true){} is a looping construct in the JavaScript programming language that allows a block of code to be repeated indefinitely. It is often used in conjunction with a break statement, which allows the loop to be exited under certain conditions.
In this case, there must be some condition inside the while loop that stops the looping otherwise browser will be crashed.

while(i < 10) { i++;(If you forget to increase the variable used in the condition, the loop will never end. This will crash your browser.) }

Do While Loop

do while loop will execute its code once(instantly for one time) whether its condition is true or not then if it is not true it stop its execution. But while loop will execute its code only if the condition is true.
do {//code; i++;} while(i < 10)

Array.forEach()

The forEach() method calls a function (a callback function) once for each array element. The forEach() method is an iterative method. It changes the array mutably.

Syntax:
Array.forEach(function(currentValue, index, arrayItself){//block of code}, thisValue);
//thisValue(optional): A value passed to the function as its this value.
