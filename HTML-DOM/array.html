Array

An objects, a function or an array may be an elements of an array.

Array is an iterable objects. Every iterable objects has Symbol.iterator method. The Symbol.iterator is a function that returns a next() function.

The iterator protocol defines how to produce a sequence of values from an object. An object becomes an iterator when it implements a next() method. The next() method must return an object with two properties:
value (the next value)
done (true or false)

See the example to understand the use of Symbol.iterator & next() method.

The typeof operator returns object because a JavaScript array is an object. To check it whether it is an array or not, use [Array.isArray(myArray)] method or [myArray instanceof Array] instanceof operator return true or false.

The Difference Between Arrays and Objects
In JavaScript, arrays use numbered indexes.
In JavaScript, objects use named indexes.
Arrays are a special kind of objects, with numbered indexes.
When to Use Arrays. When to use Objects.
JavaScript does not support associative arrays. (Arrays with named indexes are called associative arrays (or hashes). Many programming languages supports it.)
You should use objects when you want the element names to be strings (text).
You should use arrays when you want the element names to be numbers.
Uses of property and method in JS array is given below:
let myArr = ["Java", "Script", "React", "Redux"]

Frequentley Used

.every(), .some(), .filter(), .find(), .findIndex(), .forEach(), .map(), .reduce(), .reduceRight(),

1. myArr.every(function(value,index, arrItself) {}, thisValue) iterates over each array elements just like array.ForEach() but it only returns true or false.
2. myArr.filter(function(value, index, arrItself), thisValue) iterates over each array elements and returns only the filtered elements.
3. myArr.find(function(value, index, arrItself), thisValue) iterates over each array elements and returns only the first elements that matched the condition.
4. myArr.findIndex(function(value, index, arrItself), thisValue) iterates over each array elements and returns only index(position) of the first elements that matched the condition. If no matches found it returns -1.
5. myArr.forEach(function(value, index, arrItself), thisValue) iterates over each array elements and calls a function for each elements in an array. .forEach() method returns undefined.
6. myArr.map(function(value, index, arrItself), thisValue) iterates over each array elements and calls a function for each elements in an array. It returns calculated result of each element as an array.
7. myArr.reduce(function(total, value, index, arrItself), initialValue) executes a reducer function for array element(from left to right) . It returns a single value: the function's accumulated result.
8. myArr.reduceRight(function(total, value, index, arrItself), initialValue) executes a reducer function for array element(from right to left).
Note:
At the first callback, there is no return value from the previous callback.
Normally, array element 0 is used as initial value, and the iteration starts from array element 1.
If an initial value is supplied, this is used, and the iteration starts from array element 0.

Moderatley Used

.entries(), .keys(), .values(), .includes(), .indexOf(), .lastIndexOf(), .join(), .flatMap()

1. myArr.entries(); returns an iterable object with key/value pairs that can accessed by iterating over it and it doesn't change the original array. Like
for(let x of myArr.entries()) {console.log(x)}; //[0, "Java"], [1, "Script"], [2, "React"]...
2. myArr.keys(); returns an iterable object with keys(0,1,2,3) just like entries.
3. myArr.values(); returns an iterable object with values(Java,Script,React,Redux) just like entries.
Object.keys() 2.Array.from(object, mapFunction, thisValue) method returns an array from any object with a length property.
Example: Array.from("ABCD"); //["A","B","C","D"];
4. myArr.includes(element, start/position(optional)) returns true if array contains specified element.
5. myArr.indexOf(element, start/position(optional)) returns first index(position) of specified element otherwise returns -1.
6. myArr.lastIndexOf(element, start/position(optional)) returns last index(position) of specified element otherwise returns -1.
7. myArr.join("separator(sign/word/others)") makes an array into string with the separator(if used) between elements.
Example: myArr.join("*"); //Java*Script*React*Redux;
8. myArr.flatMap(function(value, index, arrItself), thisValue) maps all array elements and creates new array elements with specified condition.

Less Used

.length, .concat(), .pop(), .shift(), .push(), .unshift(), .sort(), .splice(), .slice(), .fill(), .copyWithin(), Array.from(), .isArray(), .at(), .valueOf(), .flat(), .toString()

1. myArr.length property returns the number of the elements in an array. Its also set the length of an array like, myArr.length = 2; //["Java", "Script"]
2. myArr.pop() method removes only the last element from original array and returns removed element.
3. myArr.shift() method removes only the first element from original array and returns removed element.
4. myArr.push(element1, element12,..) method add new elements to the end of the original array and returns added elements as an array.
5. myArr.unshift(element1, element12,..) method add new elements to the begining of the original array and returns added elements as an array.
6. myArr.sort() method sorts the elements in ascending(as strings in alphabetical) order and overwrites the original array. For descending order use sortedArr.reverse().
When sort() compares two values, it sends the values to the compare function, and sorts the values according to the returned (negative, zero, positive) value.
7. myArr.splice(index(position),howmany(remove), item1("add"), item2("add"),...) method add elements, remove elements or both add and remove elements from an array on specified position and returns only array of the removed elements. It also overwrites the original array.
8. myArr.slice(start, end) method returns selected elements in an array, as a new array. It does not change the original array.
9. myArr.copyWithin(1(target), 0(start), 2(end)) copies specified array elements and place them the target place like ["Java", "Java", "Script", "React", "Redux"]
10. myArr.fill(value, start, end) fills the array with the value from specified position and overwrites the original array.
11. myArr.concat(mayArr1, myArr2...) equals [myArr, myArr1, myArr2...]
12. myArr.at(index) is same as myArr[index]
