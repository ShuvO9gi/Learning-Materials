Math Related Method and Properties

Most Frequently Used

Math.abs(x) Returns the absolute value of x
Math.exp(x) means ex Returns the value of Ex
Math.floor(x) Returns x, rounded downwards to the nearest integer
Math.ceil(x) Returns x, rounded upwards to the nearest integer
Math.log(x) Returns the natural logarithm (base E) of x i.e. ln
Math.log*(x) Returns the base-*(2, 10, ...) logarithm of x
Math.cbrt(x) Returns the cubic root of x
Math.max(x, y, z, ..., n) Returns the number with the highest value
Math.min(x, y, z, ..., n) Returns the number with the lowest value
Math.pow(x, y) Returns the value of x to the power of y
Math.random() Returns a random number between 0 and 1(exclusive means maximum number will be 0.999 but not 1)
Math.round(x) Rounds x to the nearest integer
Math.PI Returns PI (approx. 3.14)
